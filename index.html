Tham số vs Đối số

  • Tham số (parameter): biến được khai báo
  • Đối số (argument): giá trị truyền vào

Composite Key cho bảng trung gian

  • Dùng object chứa 2 khóa ngoại làm ID
  • Object phải là class có 2 field FK
  • Dùng @Embeddable
  • Entity trung gian dùng @EmbeddedId
  • Dùng @MapId để map field
  • Dùng @JoinColumn để chỉ định tên cột

Instant vs LocalDateTime

  • Instant: lưu theo UTC (chuẩn quốc tế)
  • LocalDateTime: phụ thuộc timezone

Exception Structure

  • custom: định nghĩa lỗi
  • handler: xử lý lỗi

Generic Type

  • <T> đại diện cho kiểu dữ liệu
  • An toàn hơn Object

Generic Method

  • Method static phải khai báo <T>
  • Method non-static dùng trực tiếp từ class

Generic Class

public class ResponseData<T> {}

Validation Exception

  • MethodArgumentNotValidException chứa lỗi validation

UUID

  • Mapping sang MySQL: varchar(36) hoặc binary(16)

@Valid

  • Kích hoạt validation cho DTO

OOP

  • Encapsulation: đóng gói dữ liệu
  • Inheritance: kế thừa
  • Polymorphism: đa hình
  • Abstraction: trừu tượng

Abstract vs Interface

  • Abstract class: có thể chứa logic + field
  • Interface: chỉ định nghĩa hành vi

Compile-time vs Runtime

  • Compile-time: kiểm tra lỗi
  • Runtime: chạy chương trình

Collection

  • Array: cố định
  • ArrayList: resize được
  • LinkedList: dạng liên kết

List, Set, Map

  • List: có thứ tự, cho phép trùng
  • Set: không trùng
  • Map: key-value

String

  • String: immutable

Exception

  • Checked: compile-time
  • Unchecked: runtime

API Response

  • Dùng @JsonInclude(NON_NULL) để bỏ field null

DTO

  • Response cần Getter
  • Request cần Getter + Setter

React Concepts

  • useState: quản lý state

Axios

  • Gọi API

  • Khác → error

JWT

  • Xác thực giữa client và server
